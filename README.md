# Matcha Horikku - Home page

A responsive website made for serving matcha products , showing general info about matcha , our top selling products , a shop by mood section and more to add later on.

## Features
- Responsive design 
- Sticky sidebar for easy navigation
- Glassmorphism effects
- Material UI for styling

## Tech Stack
- React
- Vite
- Material UI
- CSS

## Installation

```bash
git clone https://github.com/yourusername/matcha-horikku.git
cd matcha-horikku
npm install
npm run dev
```
